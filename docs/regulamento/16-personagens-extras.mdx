

# 16. Personagens Extras

Cada jogador pode ter múltiplos personagens, desde que todos sigam regras rígidas para evitar metagaming, confusão de histórias ou vantagens injustas.

---

## 16.1 Regras gerais

<Check>
  **Você deve:**

  <Check>
    Criar histórias distintas para cada personagem.
  </Check>
  <Check>
    Evitar qualquer conexão IC entre personagens.
  </Check>
  <Check>
    Manter comportamentos, roupas e personalidades separadas.
  </Check>
  <Check>
    Fazer RP coerente com cada identidade.
  </Check>
</Check>

---

## 16.2 O que é proibido

<Danger>
  **Proibido:**

  <Danger>
    Usar informações de um personagem no outro.
  </Danger>
  <Danger>
    Fazer negócios entre seus próprios personagens.
  </Danger>
  <Danger>
    Ajudar seu personagem principal com o secundário.
  </Danger>
  <Danger>
    Usar personagem extra para farmar itens.
  </Danger>
  <Danger>
    Criar personagem apenas para “se livrar” de enrascadas.
  </Danger>
</Danger>

---

## 16.3 Facções e empregos

<Warning>
  Cada personagem deve ter **sua própria vida**:
  <Tip>
    Profissão diferente  
  </Tip>
  <Tip>
    Contatos distintos  
  </Tip>
  <Tip>
    Trajetórias separadas  
  </Tip>
</Warning>

<Warning>
  Proibido ter dois personagens ativos em facções diferentes ao mesmo tempo.
</Warning>

---

## 16.4 Consequências

<Warning>
  Violações relacionadas a personagens extras resultam em:

  <Warning>
    Perda do personagem
  </Warning>
  <Warning>
    Confisco de bens
  </Warning>
  <Warning>
    Suspensão
  </Warning>
  <Warning>
    Banimento (em casos graves)
  </Warning>
</Warning>
